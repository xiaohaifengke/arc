> Generate arcs routes as lines that does not coincide between to points.

## Install
npm
```shell
npm i arc-points -S
```
or yarn
```shell
yarn add arc-points
```

## Usage
Require the library in node.js like:

```javascript
var arcPoints = require('arc-points');
```

Use in the browser like:

```html
<script src="./arc-points.js"></script>
```
